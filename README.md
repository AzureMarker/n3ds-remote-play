# Nintendo 3DS Remote Play
A project to stream PC games to the 3DS.

[![Build status](https://badge.buildkite.com/3407926df3a48a8ec88800d2025061ac9b79cdb06002135602.svg)](https://buildkite.com/mark-drobnak/n3ds-remote-play)

Features (work in progress):
- [x] Stream 3DS inputs to the PC as a virtual controller.
- [x] Stream PC video to the 3DS.
- [ ] Stream PC audio to the 3DS.
